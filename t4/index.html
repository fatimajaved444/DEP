My Projects

Candu Crush

"I created a Candy Crush-like game using C++ to replicate the engaging mechanics of match-3 puzzle games. This project involved designing a grid-based game board, implementing match detection algorithms, and handling user input to create a dynamic and interactive gameplay experience. By focusing on game logic, graphics, and animations, I developed a comprehensive understanding of game development principles and object-oriented programming. This project allowed me to refine my skills in C++ and explore the complexities of game design, resulting in a fun and challenging game that demonstrates my ability to tackle intricate programming tasks."
